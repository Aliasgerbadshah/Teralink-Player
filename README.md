# Teralink-Player

A fast, modern web-based video player to extract and stream TeraBox `.m3u8` video links natively using HLS directly in your browser without CORS limitations. Uses a local PHP proxy or lightweight Node.js proxy to connect to external endpoints like xAPIverse.

## Setup
Upload these files to any PHP server or run locally using Node.js (`node server.js`). 
Open `index.html` in your browser.
